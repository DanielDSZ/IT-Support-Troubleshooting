
# Preparação de Máquina

Este material serve como um procedimento padrão (SOP) para preparar e entregar uma máquina corporativa do jeito “pronto pra produção”, usando servidor de imagem (FOG Project) para fazer o deploy via rede (PXE) e depois executar os ajustes finais.

Ele cobre o fluxo completo: validar o chamado e periféricos, configurar BIOS (boot via rede/PXE, Secure Boot, senha), dar boot na rede e aplicar a imagem, renomear o equipamento no padrão da empresa, rodar Windows Update + drivers, instalar antivírus, entrar no domínio ou criar usuário local conforme o cenário (sede x fora da sede), validar apps essenciais (Office/VPN) e finalizar com limpeza, etiquetagem e embalagem para envio/expedição.
