
# Servidor Imagem (Fog)

Este material apresenta um passo a passo para implantar e operar um servidor de imagem com o FOG Project, focado em padronizar a instalação de sistemas em múltiplas máquinas via PXE. Ele cobre desde a preparação do servidor (requisitos de hardware/armazenamento e SO base), configuração de IP fixo e boas práticas de rede (ex.: VLAN dedicada para isolar tráfego de imaging), até a preparação da “golden image” (imagem genérica + Sysprep), o deploy em novos equipamentos e o pós-deploy (drivers, reativar Secure Boot, desabilitar boot por rede e senha na BIOS/UEFI) para fechar o ciclo com segurança e consistência.
