
# Backup Outlook

Este material é um guia rápido e visual para fazer backup do Microsoft Outlook, exportando os dados da caixa de e-mail (ou pastas específicas) para um arquivo .pst. Ele mostra o caminho no menu do Outlook (Arquivo → Abrir e exportar → Importar/Exportar), como selecionar a conta/pasta, configurar opções como itens duplicados e período do backup, e por fim escolher um diretório seguro (com recomendação de salvar em Documentos e, se possível, sincronizado no OneDrive) — incluindo a ideia de exportação recorrente para manter o .pst sempre atualizado.
