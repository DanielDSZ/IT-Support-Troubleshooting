
# Mapeamento de Rede

Este material é um guia rápido para mapear uma unidade de rede no Windows (criar um “drive” tipo I: ou Z: apontando para uma pasta em um servidor via caminho UNC \\servidor\pasta). A ideia é dar acesso fácil e consistente à sua pasta de rede (ex.: pasta do seu login), com a opção de reconectar automaticamente ao entrar e autenticar usando domínio\usuário quando solicitado.
